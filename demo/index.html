viridis

The viridis colormap is perceptually uniform, colorblind-friendly, and monotonically increasing in lightness — which is why it became the default way to color data.

This page demonstrates the @graysonlang/viridis npm package — a tiny, dependency-free ES module bundling viridis and its siblings. Pick a colormap at the top right and every visualization follows.

npm install @graysonlang/viridis

Colormap data: the viridis family by Stéfan van der Walt and Nathaniel Smith (matplotlib, CC0); cividis by Nuñez, Anderton & Renslow; turbo by Anton Mikhailov (Google); mako and rocket by Michael Waskom (seaborn).

This page bundles viridis.js plus d3-delaunay for the Voronoi tile.
